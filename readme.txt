MBT Testimonial is a fully Responsive & mobile friendly WordPress plugin to manage your client Testimonials.

== Description ==

MBT Testimonial Slider is a fully Responsive & mobile friendly WordPress plugin to manage your client Testimonials.

How To Use It.

Default shortcode is [mbt_testimonial]

If you make category then you need to mention category name

[mbt_testimonial category="abc"]

== Installation ==

Just install from your WordPress "Plugins > Add New" screen and all will be well. Manual installation is very straightforward as well:

1. Upload the zip file and unzip it in the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to MBT Testimonial to create testimonial slides.
